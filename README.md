# Website Profile

Latihan membuat website profile sederhana dengan HTML dan CSS3.

## Syntax Awal
Menambahkan `README.md` untuk commit pertama.

```bash
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:manjokinara/manjokinara.github.io.git
git push -u origin main
```

## Syntax Lanjutan
Menambahkan file lainnya dengan `.` (seluruh file) ke github.

```bash
git add .
git commit -m "update files"
git push
```
